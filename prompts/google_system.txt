## Code Review Focus Areas

Review this codebase for common issues:

1. **Input Handling:** Edge cases, boundary conditions, error handling
2. **Memory Management:** Resource leaks, proper cleanup, bounds checking
3. **Error Handling:** Unchecked returns, missing null checks, exception handling
4. **Logic:** Off-by-one errors, incorrect conditions, state management
5. **Concurrency:** Race conditions, deadlocks, thread safety

**Output Format:**
For each issue found:
- Title describing the bug
- Severity (Critical/High/Medium/Low)
- Affected file and line number
- Clear steps to reproduce
- Impact if not fixed
- Suggested fix

Focus on finding real bugs that would cause incorrect behavior.
